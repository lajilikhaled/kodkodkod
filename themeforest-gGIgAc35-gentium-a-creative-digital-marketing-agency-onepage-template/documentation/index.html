File structure of the source

Our template contains all source files which are included into the assets folder. Sass sources are compiled into the main CSS file style.css. and there is a Scss folder included a Sass files allows you to include customizations in the build process, rather than manually overwriting a lot of CSS rules by hand.

Folder Description
/assets/css Contains all needed CSS files.
/assets/js Contains all JavaScript files included the main.js initial file.
/assets/fonts Contains a custom icons font Pixeicons
/assets/images Contains all image sources.
/scss Contains sass files which not compiled yet to CSS, updated on every release.
/includes Contains functional files.

Stylesheets

Here's a list of the stylesheet files we are using with this template, you can find more information by opening each file:

File Name Description
style.css The main CSS file of this template, Rules like headings and paragraphs, margins and body and more...
uikit.min.css UIkit 3 Framework CSS file.
normalize.min.css Makes browsers render all elements more consistently and in line with modern standards.
owl.carousel.min.css Rules for Sliders.
pr.animation.css Rules for animations.
pixeicons.css Rules for font icons.
fonts.css Rules for custom font types.

JavaScript

Here's a list of the JavaScript files we are using with this template, you can find more information by opening each file:

File Name Description
jquery.min.js JavaScript UI library.
uikit.min.js Uikit3 Framework JavaScript file.
owl.carousel.min.js A jQuery plugin for Owl Carousel Slider.
anime.min.js A jQuery plugin for interactive design or animation.
pr.animation.js This file pertains to everything related to interactive design or animation. you can delete it if you don't need animations. Is compatible with anime.js jQuery plugin.
validate.js A jQuery plugin for validate inputs on contact form.
main.js All custom scripts are initialized from this file.

PHP

Here's a list of the php files we are using with this template, you can find more information by opening each file:

File Name Description
